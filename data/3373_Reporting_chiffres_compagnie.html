Reporting chiffres compagnie

Les reportings sont à communiquer au Directeur Transport et réalisés par les Transport Operations Manager ## Chiffres compagnies BSP En début de mois (premier jour travaillé), sur la base du BSP, les chiffres de toutes les compagnies aériennes sont reportées sur la tableau excel intitulé “Suivi compagnies BSP”, disponible sur Wrike. Une tâche Wrike est donc programmée pour l’ensemble des Transport Operations Manager. ## Chiffres Emirates Le tableau de suivi des ventes Emirates est complété par les Transport Operations Manager. Ce tableau est disponible sur Wrike. Une tâche Wrike est donc programmée pour l’ensemble des Transport Operations Manager. +-Toutes les semaines - détail des ventes Si l’activité le permet, il est possible de renseigner le tableau tous les jours. Toutes les semaines, l’onglet “Détail des ventes” est complété par un agent. Cet onglet donne dans le détail toutes les ventes réalisées par des agences de voyages qui émettent avec nous leurs dossiers Emirates (en resa IATA). Pour le compléter : 1. Lancer la recherche dans Galaxy avec les filtres : - Étiquette : Resa IATA - Date de réservation : la semaine précédente du lundi au dimanche - Compagnie : EK 2. Compléter le tableau avec les informations suivantes, sur la base du dossier Galaxy: - Nom de l’agence (sélectionner dans la liste) - Mois d’émission du billet (Ex : mettre au 01/04/2016 pour avril) - Prix TTC - Le nombre du pax 3. Aller dans le BSP Link, rubrique consultation générale/analyse de facturation en ligne et compléter le formulaire en mettant : - Code compagnie : 176 (Emirates) - Période du BSP : période(s) correspondant à la semaine en question /!\ aux changements de période 4. Retrouver dans la liste, les billets correspondant aux ventes précédemment enregistrées dans le tableau et renseigner le montant Hors Taxes dans le tableau. +-Tous les mois - reporting mensuel Cet onglet donne dans le détail toutes les ventes réalisées par des agences de voyages qui émettent avec nous leurs dossiers Emirates (en resa IATA). Le premier jour travaillé de chaque mois : réaliser le reporting mensuel, basé sur l’onglet du détail des ventes hebdomadaire et mettre à jour l’onglet “classement des agences”. - Chiffres mensuels - Classement agences 1. Sélectionner l’ensemble du tableau et aller dans le menu Données/tableau croisé/créer 2. Compléter les champs comme ci-dessous, puis cliquer sur OK. - Agence => champs de ligne - Date d’émission => champs de colonne - Nombre de pax => champs de données - Tarif TTC => champs de données - Tarif HT => champs de données. [Image: Tableau croisé | http://gravity.vlctravel.com/wp-content/uploads/2016/08/Tableau-croisé.jpg] 3. Cela génère un nouveau tableau, dans le même onglet. 4. Copier la colonne du mois désiré et faire un collage spécial pour ne reporter que les nombres (sans mise en forme) sur l’onglet “chiffres mensuels”, y compris les totaux. 5. Sur l’onglet “chiffres mensuels”, reporter le montant total des billets Emirates émis le mois précédent (HT puis TTC), selon le BSP. Les chiffres sont reportés automatiquement. Il faut simplement mettre à jour le classement. 1. Sélectionner le tableau (en dehors de l’en tête et de la colonne A) puis aller dans le menu données/trier. 2. Dans critère de tri/ Tri selon mettre : - Colonne F - Décroissant Les agences seront triée automatiquement par volume de vente HT.
