Etats de caisses (Daily Task)

Mise à jour le 09/12/2021
Mise à jour le 09/12/2021 Daily Task à faire tous les matins dès l’ouverture. Le pointage des états de caisse est réalisé quotidiennement par le service administratif. Dans certain cas, cela peut prendre plusieurs heures. C’est pourquoi il est nécessaire que le service Exec édite rapidement les états de caisse. Pour chaque GDS, 1 – Compléter le fichier papier de suivi, en additionnant les états de caisses EMD et BILLETS à chaque fois. 2 – Enregistrer l’état de caisse dans le dossier correspondant sur PENGUIN_WORLD_GLOBAL/VLC_TRAVEL/RESA/Etat de Caisse +-Amadeus 1. Trois formats à faire, l’un pour le montant total, l’autre pour le détail billet/billet, le dernier pour le Rail WM/FWD/EML booking@vlctravel.com/TJD/SOF/D-date WM/FWD/EML booking@vlctravel.com/TJQ/SOF/D-date WM/FWD/EML booking@vlctravel.com/TJQ/A-FRR/SOF/D-date 2. Copier coller les données reçues par mail dans un seul fichier texte et l’enregistrer dans le dossier correspondant aux GDS PENGUIN_WORLD_GLOBAL/VLC_TRAVEL/RESA/Etat de Caisse +-Sabre 1. Deux formats à faire, l’un pour le montant total, l’autre pour le détail billet/billet DQB*date DQB*EMD/D date 2. Copier coller les données dans un seul fichier texte et l’enregistrer dans le dossier correspondant aux GDS PENGUIN_WORLD_GLOBAL/VLC_TRAVEL/RESA/Etat de Caisse +-Galileo France et Italie L’action doit être réalisée deux fois, l’une sur PCC 5D4X, l’autre sur PCC 3T4N 1. Un seul format à faire pour obtention du total : EMD + BILLETS HMPR/date NB : en faisant ALT+C il est possible d’afficher l’état de caisse dans sa totalité 2. Copier coller les données dans un seul fichier texte et l’enregistrer dans le dossier correspondant aux GDS PENGUIN_WORLD_GLOBAL/VLC_TRAVEL/RESA/Etat de Caisse Particularité : Sur le document à remplir manuellement, deux colonnes sont indiquées BSP et CB.
