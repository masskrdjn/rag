Date de la dernière mise à jour : 22/03/2024 Pour générer une CB via NIUM : - Choisir la CB : mc_200_sing et funding_account : MastercardEUR [Image: image | http://gravity.vlctravel.com/wp-content/uploads/2024/02/CB1-1.png] - Remplir la partir “Additionnal Details” comme suit : en #1 dans Name : ref et dans Value le numéro de résa SpeedTravel en #2 dans Name : achat et dans Value le montant prévu d’achat en EUR (celui de STRA , avec un point pour séparer les décimales et 2 décimales) en #3 dans Name: canalID et dans Value 1 en #4 dans Name: connecteurID et dans Value 1 en #5 dans Name: carrier et dans Value FR Voici un exemple  : [Image: image | http://gravity.vlctravel.com/wp-content/uploads/2024/02/CB3.png]                   Générer la CB une fois tous les éléments remplis.
