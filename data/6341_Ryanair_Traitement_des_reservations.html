Ryanair – Traitement des réservations

Dernière mise à jour : 23/01/2025 Les réservations Ryanair nécessite une intervention manuelle pour la confirmation du dossier. ## Etape 1 : réception du dossier en “inconnue” dans l’onglet URGENCES sur galaxy Le dossier arrive dossier en “inconnue” dans l’onglet URGENCES sur galaxy. A ce stade, le dossier est confirmé pour le client et le dossier Galaxy est déjà remonté dans nos systèmes. Toutefois, attention, le dossier n’est pas confirmé auprès de Ryanair, il faut faire la réservation sur Galileo. ## Etape 2 : Réservation du dossier auprès de Ryanair  +-Avec Travelport/Galileo Vous trouverez ici le TUTO pour effectuer la réservation. Si vous souhaitez des informations sur les différentes classes de réservation disponibles, Cliquez ici . La réservation doit être payée en carte bancaire virtuelle, générée via Airplus.  Quelques conseils toutefois : - Ne pas oublier d’insérer la civilité au moment d’entrer les noms et prénoms des passagers sur galileo : N.NOM/PRENOM MR - Si votre réservation contient un CHD, voici le format pour entrer les infos pax : N.NOM/PRENOM MR*P-C10  ; il faudra également insérer les SR DOCS pour ce passager. - Si votre réservation contient un INF, voici le format pour entrer les infos pax :  N.I/NOM/PRENOM MS*05MAR22 ; il faudra également insérer les SR DOCS pour ce passager. - Attention, une fois les infos de carte bancaire renseignées dans le PNR, si vous faites ER > vous payez et confirmez le dossier - Le plus avantageux en terme de coût et de prendre un tarif BASIC (Y) et d’ajouter ensuite les prestations supplémentaire avec le caddie. - Si vous constatez une différence tarifaire impliquant une marge négative > Informez votre responsable - Mettre l’adresse mail générée dans le suivi Galaxy comme adresse mail du dossier ainsi que le numéro du client (trouvable dans l’onglet PAX sur Galaxy)   ## /!\ Je n’arrive pas confirmer mon dossier sur Galileo : quelles sont les solutions? /!\ - Message d’erreur au moment de la confirmation du dossier  : FR – AGENT IS NOT AUTHORIZED TO OVER PAY ON A BOOKING Si vous rencontrez ce message au moment de la confirmation, essayer de réserver en réservant l’aller et le retour séparément. (sur 2 PNR différents) - Si la carte est rejetée par le système : Alertez votre responsable pour obtenir une carte de secours (AMEX ou autre). Si malgré cette nouvelle carte, le dossier ne se confirme toujours pas : tenter de reprendre sur le site de Ryanair, puis Trip.com en dernier recours. - Je ne trouve pas les vols sur Galileo Si les vols ne sont pas trouvables sur Galileo, en ayant essayer de recherche le vols par la FS ou par l’affichage de dispo, tenter de reprendre sur le site de Ryanair, puis Trip.com en dernier recours. +-Avec Speedtravel Dans l’onglet “Reservation” sur SpeedTravel, cliquer sur “nouvelle commande” : - Sélectionner le canal “Repro Resaneo” et indiquer les éléments du voyages (date/destinations/nombre et type de pax) [Image: image | http://gravity.vlctravel.com/wp-content/uploads/2023/11/repro-RN.png] - Sélectionner le fournisseur 1G – ACH (cf ci-dessous)  et envoyer la recherche. [Image: image | http://gravity.vlctravel.com/wp-content/uploads/2023/11/Repro-FR-STRA.png] [Image: image | http://gravity.vlctravel.com/wp-content/uploads/2023/11/zj27k4h5cDOhtIBc.png] - Saisissez les prestations à ajouter : Embarquement prioritaire ; Bagages - Renseignez les informations clients en indiquant bien les coordonnées passagers (numéro de téléphone des passagers, onglet PAX su GX) - Insérez l’adresse mail de réservation de la résa initiale (dans le suivi GX) au format booking-xxx@resatravel.net  : [Image: image | http://gravity.vlctravel.com/wp-content/uploads/2023/11/DaA3RcEQagKRlMYE.png] - Une fois tous les champs renseignés, validez la réservation. - Dès validation, il faut mettre à jour les informations dans Galaxy (numéro de PNR, numéro de la REPRO STRA en note dans le suivi) En cas de bug, alertez immédiatement votre responsable. +-Sur le site Ryanair 1 – Faire la simulation sur le site Ryanair 2 – Dans le processus de réservation, le système de Ryanair va vous demander de vous connecter. Nous vous connectez pas ! A la place, il faut créer un nouveau compte en insérant l’adresse mail initiale présente dans le suivi de dossier ou bien si il n’y en a pas il faut la créer manuellement en fonction de la référence de votre dossier : booking-GX000000@resatravel.net Par exemple, si vous êtes sur le GX123456, vous insérez l’adresse mail : booking-GX124536@resatravel.net Tout les mails qui seront envoyés à cette adresse mail arriveront sur l’adresse mail booking@. 3- Une fois cette étape effectuée, vous pouvez remplir le formulaire avec les coordonnées de la personne en charge de la réservation : Prénom : François / Nom : XAVIER / Adresse postale : 73 ter rue Francis de Présensé 69100 VILLEURBANNE Et définir un mot de passe. N’oubliez pas d’utiliser au besoin : https://www.motdepasse.xyz/ (Gardez le mail et le mot de passe pour pouvoir les mettre en note dans le dossier GX)  Et ainsi continuez jusqu’à la page de paiement. 4 – Pour procéder au paiement il faudra générer une carte via Galaxy : rapprochez vous d’un Team-Leader ou un OPM si la CB galaxy n’a pas fonctionné pour vous générer une carte Airplus. 5 – Une fois la réservation effectuée et le paiement validé, insérez le mail de réservation et le mot de passe en note dans votre dossier Galaxy. C’est très important pour pouvoir assurer le SAV plus tard. 6 – N’oubliez pas d’envoyer les infos pour les Check’In au client ! +-Sur Amadeus A utiliser principalement pour les dossers FR/RK qui ne passe pas dans les 3 solutions proposées juste avant.
